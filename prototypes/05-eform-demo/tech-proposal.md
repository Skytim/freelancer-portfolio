## 技術棧

| 層級 | 技術 | 原因 |
|---|---|---|
| 前端 | React 18 + Vite + Tailwind | 動態表單渲染，PWA 支援成熟 |
| 離線 | IndexedDB (Dexie.js) + Service Worker | 地下機房沒訊號也能填，回連線後自動同步 |
| 後端 | Node.js (Express/Fastify) 或 Python (FastAPI) | 輕量，前後端統一 JS 生態 |
| 資料庫 | PostgreSQL (生產) / SQLite (地端備援) | 關聯性強（設備↔工單↔照片） |
| 照片 | AWS S3 或 MinIO (地端) | 照片量大，物件儲存成本低 |
| PDF | html2pdf.js (前端直接產生) | 免後端渲染，離線也能輸出 |
| QR Code | 前端 `qrcode.js` + 後端標籤列印 | 設備 QR 貼紙可從後台批次列印 |
| 部署 | Docker + Nginx 反向代理 | 一鍵部署，雲端地端通用 |

## 雲端 vs 地端建議

| | 雲端（AWS / GCP） | 地端（客戶自有伺服器） |
|---|---|---|
| 適合情境 | 多案場、跨區域 | 單一大型機房、資安要求高 |
| 建置成本 | 低（月費 $2,000-5,000） | 中（需自備主機） |
| 維護 | 我遠端維護 | 需定期到場 |
| 離線需求 | PWA + 定期同步 | 內網 + PWA 備援 |
| **建議** | ✅ **MVP 先用雲端**，後續可擴充地端備援 | |

## 動態表單引擎設計

這是本案核心挑戰——讓後台可自訂表單欄位。

```
表單模板 (JSON Schema)：

{
  "template_id": "ups_inspection",
  "name": "UPS 巡檢表",
  "fields": [
    { "key": "input_voltage", "label": "輸入電壓", "type": "number", "unit": "V", "required": true },
    { "key": "output_voltage", "label": "輸出電壓", "type": "number", "unit": "V", "required": true },
    { "key": "battery_status", "label": "電池狀態", "type": "select", "options": ["正常","需注意","異常"] },
    { "key": "visual_check", "label": "外觀檢查", "type": "checkbox", "items": ["外殼無破損","風扇正常","無異音"] }
  ]
}
```

後台介面可直接新增/編輯欄位，前端 React 根據 Schema 動態產生表單。
