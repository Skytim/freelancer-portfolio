## 工程師核心策略

### 你的賣點不是設計，是技術

```
設計師的武器：美感、配色、排版、品牌調性
你的武器：SEO 架構、結構化資料、網站速度、後端邏輯、自動化
```

**所有需要大量視覺設計的案子，對你來說都是誤區。** 即使勉強接了，耗時會是設計師的 3 倍，品質還更差。

### 設計問題的解法

當案子的設計需求無法迴避時，三條路：

| 策略 | 適用 | 成本 |
|---|---|---|
| 買 WP 主題套版 | 診所、形象站、批發零售 | $1,500~2,500 一次（ThemeForest） |
| 用電商平台版型 | 品牌電商 | $0，平台內建的 premium 版型 |
| 外包給設計師 | 高度客製需求 | $5,000~8,000/案，從預算扣 |

## 各案優劣分析

### 1. 形象網站 — 桃園桃園區

| 預算 | 期限 | 技術 | 設計佔比 |
|---|---|---|---|
| 3 萬以下 | 未指定 | 未指定 | 70% |

**優點**
- 技術門檻低，靜態網站 + RWD 即可
- 範圍相對單純（3~5 頁面）

**缺點**
- 核心價值在視覺設計，不是你的強項
- 預算太低，含設計+切版+後台，時薪不到 400 元
- 設計來回修改的隱形成本沒有計入

**工程師建議**：除非急著要第一個作品，否則跳過。如需承接，用 WP + 現成主題，不要從零設計。

### 2. 網站 + 後台管理系統 — 桃園中壢區

| 預算 | 期限 | 技術 | 設計佔比 |
|---|---|---|---|
| 5~10 萬 | 一般件 | Python / PHP | 30% |

**優點**
- 預算比第一案合理
- 後端技術棧主流（Python/PHP），Django Admin 或 Laravel Filament 可快速交付後台
- 後台不需設計美學，功能為王

**缺點**
- 「網站+後台」範圍可以差 10 倍，需仔細釐清
- 業主同時列 Python 和 PHP，可能不懂技術，溝通成本高
- 前端設計仍需解決（用主題）

**工程師建議**：可接但必須限縮。報價鎖上限、功能明確列在合約、分階段收款。用 Django Admin 打後台，前端買主題。

### 11. WP 醫療診所 SEO/AIEO 升級 — 桃園中壢區 ⭐ 首選

| 預算 | 期限 | 技術 | 設計佔比 |
|---|---|---|---|
| 3 萬以下 | 越快越好 | WordPress | 10% |

**🟢 軟體工程師最適合的第一案**

**優點**
- 🔑 核心價值是 SEO 技術（Schema、llms.txt、PageSpeed），全是你的賽道
- 🔑 診所網站不需要美學設計——白底藍字、乾淨整齊就夠了
- WP 操作為主，選單調整、SEO 設定
- AIEO（AI Engine Optimization）是新興利基技能，能成為差異化賣點
- 長期文章維護 = 每月被動收入（5,000~8,000/月）
- 醫療業客戶忠誠度高，不太會換廠商
- 三個這類客戶就能打底月收入

**缺點**
- 一次性收入偏低（3 萬以下），真正價值在後續維護
- 需學習 SEO/AIEO 相關知識（llms.txt、Schema 結構化資料）

**工程師策略**：
- 設計部分：花 $1,500~2,500 買 ThemeForest 醫療診所 WP 主題，套上去微調就好
- 核心交付：SEO 技術審查報告 + Schema 部署 + llms.txt + Google Search Console 串接
- 提案時用 prototype（已做好）展示 SEO 技術能力，不是展示設計

**營收模式**：
| 收入類型 | 金額 | 頻率 |
|---|---|---|
| 一次性 SEO 設定 + 選單調整 | 2~3 萬 | 一次 |
| 持續性：文章上架 + 維護 | 5,000~8,000/月 | 每月 |
| 擴充：關鍵字規劃、競品分析 | 另報 | 季 |

**關鍵提問清單**：
1. 現有網站網址？

3. 「改版更美觀」有參考網站嗎？（用來挑主題方向）
4. 「新功能」具體指什麼？（這才是你收費的主力）
